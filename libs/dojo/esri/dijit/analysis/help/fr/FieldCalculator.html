Calculatrice de valeurs de champs

Calcule les valeurs d'un champ (attribut) en fonction d'une formule que vous spécifiez.

Expressions

Vous pouvez créer des expressions pour calculer les valeurs de champs (attributs) nouveaux ou existants.

Après avoir sélectionné un champ, vous pouvez créer une expression en cliquant dans la zone Expression pour lancer le Générateur d’expressions. Dans le Générateur d'expressions, une expression peut être créée à partir de n'importe quelle combinaison d'autres champs, opérateurs numériques, fonctions Python et valeurs numériques et de chaîne. Pour calculer les valeurs d'un nouveau champ, vous devez définir les propriétés du champ, notamment l'alias, le type de champ et, pour les attributs textuels, la longueur du texte que le champ contiendra.
