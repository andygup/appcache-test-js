Calculadora de campo

Calcula los valores de un campo (atributo) en función de la fórmula que especifiques.

Expresiones

Puedes crear expresiones para calcular los valores de campos (atributos) nuevos o existentes.

Después de seleccionar el campo, la expresión se crea haciendo clic en la casilla de expresión para abrir el Constructor de expresiones. En el Constructor de expresiones, se puede crear una expresión a partir de cualquier combinación de otros campos, operadores numéricos, funciones Python, valores numéricos y cadenas de caracteres. Si estás calculando los valores de un campo nuevo, debes definir las propiedades del campo incluyendo el alias, el tipo de campo y, para los atributos de texto, la longitud del texto que contendrá el campo.

Nombre de capa de resultados

Este es el nombre de la capa que se creará en Mi contenido y se agregará al mapa. El nombre predeterminado se basa en el tipo de capa de análisis. Si la capa ya existe, se te pedirá que confirmes si deseas sobrescribirla.

En el cuadro desplegable Guardar resultado en, puedes especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado.
