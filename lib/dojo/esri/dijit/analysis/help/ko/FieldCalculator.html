필드 계산기

지정한 수식을 기반으로 필드(속성)의 값을 계산합니다.

식

새 필드(속성) 또는 기존 필드(속성)의 값을 계산할 식을 생성할 수 있습니다.

필드를 선택한 후 식 상자를 클릭하면 실행되는 식 작성기를 사용하여 식을 작성할 수 있습니다. 식 작성기에서 다른 필드, 숫자 연산자, Python 함수 및 숫자/문자열 값의 조합을 통해 식을 생성할 수 있습니다.새 필드의 값을 계산하는 경우 별칭, 필드 유형 및 필드에 포함할 텍스트의 길이(텍스트 속성의 경우)가 포함된 필드의 등록정보를 정의해야 합니다.

결과 레이어 이름

내 컨텐츠에서 생성되어 맵에 추가되는 레이어의 이름입니다.기본 이름은 분석 레이어 이름을 기반으로 합니다.레이어가 이미 있는 경우 덮어쓸지 확인하는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 컨텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
