Pode criar expressões para calcular valores de campos (atributos) novos ou existentes.

Após a seleção de um campo, é possível construir uma expressão clicando na caixa de expressão para iniciar o Expression Builder. No Expression Builder, é possível criar uma expressão a partir de qualquer combinação de outros campos, operadores numéricos, funções Python e valores numéricos e de strings. Se pretender calcular valores de um novo campo, tem de definir as propriedades do campo, incluindo o nome alternativo, o tipo de campo e, para atributos de texto, o comprimento do texto contido no campo.

Nome da camada de resultado

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, ser-lhe-á pedido que confirme se pretende substituir a camada.

Utilizando a caixa de lista pendente Guardar resultado em, pode definir o nome de uma pasta em O Meu Conteúdo para guardar o resultado.
